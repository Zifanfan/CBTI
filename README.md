# CBTI — Chinese-Based Temperament Inventory

> 中国特色人格测试。十个问题，四个字母，勾勒你看待世界的方式。

🔗 **在线体验：** [zifanfan.github.io/CBTI](https://zifanfan.github.io/CBTI/)

---

## ✨ Features

- **10 道情境题** — 贴近中国年轻人真实生活的场景（过年纹身、外卖迟到、朋友圈热点、职场羊排博弈……）
- **11 种道德/社会人格** — 每种人格配有直觉性的 4 字母代码：

  | Code | 人格名称 | 核心驱动 |
  |------|---------|---------|
  | `LULZ` | 纯粹解构乐子人 | 趣味与解构 |
  | `CALC` | 精致理性利己者 | 个人利益最大化 |
  | `EMPT` | 感性共鸣优先者 | 感性共鸣 |
  | `VOID` | 彻底虚无躺平者 | 无意义感 |
  | `JUST` | 原则至上理想者 | 普世真理与正义 |
  | `GRND` | 绩效至上攀登者 | 成功与阶层跃升 |
  | `NARR` | 宏大叙事信仰者 | 集体荣誉与身份认同 |
  | `PHIL` | 独立思辨哲学家 | 理解与内在完整 |
  | `RULE` | 规则秩序守护者 | 稳定与规则 |
  | `COZY` | 具体幸福生活家 | 具体的幸福 |
  | `UNDF` | 不被定义漫游者 | 主体性与疏离 |

- **加权累积评分算法** — 每个选项同时向 1~3 种人格类型加分（主倾向 +5，次倾向 +2~3），最终得分最高者为你的人格类型
- **隐藏第 11 人格** — 每道题都有第 5 个选项「ABCD都不对，我有自己的想法」，选择它会为 `UNDF`（不被定义）加分
- **可回退修改** — 支持随时返回上一题查看和修改答案
- **选中自动跳转** — 选择选项后 150ms 自动进入下一题，流畅无打断
- **结果页完整展示** — 人格代码 + 中文名称 + 详细描述 + 全维度得分分布图 + 每题选择回顾
- **纯静态单文件** — 一个 `index.html` 搞定一切，无需后端，无需构建，浏览器直接打开即可运行
- **暗色文艺风 UI** — 深藏蓝背景 + 暖琥珀色调 + Noto Serif SC 衬线字体 + 圆点进度指示 + 柔和动画

## 🚀 使用

直接用浏览器打开 `index.html`，或部署到任意静态托管平台（GitHub Pages、Netlify、Vercel 等）。

## 🛠 自定义

编辑 `index.html` 中的 `quizData` 数组即可修改题目、选项和评分权重。编辑 `TYPE_INFO` 对象即可修改人格类型的名称、代码和描述。
